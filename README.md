# Certus Index Spline
Open-source module for robust, ultra-broadband spectrophotometric reverse engineering of dielectric thin films.

## Overview
This repository contains the numerical core of the Certus Index Spline algorithm, allowing for the simultaneous extraction of the complex refractive index ($n$, $k$) and the physical thickness $d$ of thin films. It uses a dynamic, multi-node spline model with a sequence-gated optimization workflow.
